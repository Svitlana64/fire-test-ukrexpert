# **🔥 Інтерактивний експерт-тест «Інструктаж з пожежної безпеки: який, коли, навіщо»**

Сучасний, адаптивний та футуристичний веб\-тест, розроблений спеціально для **Науково-виробничого центру "Укрексперт"**. Тест дозволяє керівникам, відповідальним особам та працівникам перевірити свої знання юридичних та організаційних нюансів проведення протипожежних інструктажів.

Всі питання та відповіді базуються на актуальних нормативно-правових актах України:

* 📜 **Порядок затвердження програми інструктажів № 1021**  
* 📜 **Правила пожежної безпеки в Україні № 1417**

## **✨ Особливості проекту**

* 🛸 **Футуристичний діджитал-дизайн**: Глибокий фірмовий фоновий градієнт на основі кольору \#18505b з пастельними світло-блакитними акцентами \#abe2ea, неоновим світінням та технологічною цифровою сіткою.  
* 📱 **Адаптивна верстка (Mobile-First)**: Інтерфейс однаково бездоганно виглядає як на великих моніторах комп'ютерів, так і на екранах смартфонів чи планшетів.  
* 🧠 **Експертний фідбек**: Після кожної відповіді користувач миттєво бачить, чи відповідь правильна, а також отримує розгорнутий коментар-пояснення від експертів НВЦ "Укрексперт".  
* 📊 **Аналітика результатів**: Наприкінці тестування виводиться загальний бал та персональний вердикт щодо ризиків перевірки ДСНС на підприємстві, а також детальний покроковий аналіз усіх питань.  
* 🔗 **Інтеграція з Facebook (Open Graph)**: У код вбудовані мета-теги для красивого шерингу в соцмережах (відображається приваблива картка з прев'ю вашого 3D-логотипу).

## **📂 Структура проекту**

Репозиторій складається всього з двох основних файлів, що робить його запуск максимально швидким та легким:

1. index.html — інтерактивний додаток, що містить всю HTML-розмітку, логіку на JavaScript та стилі Tailwind CSS.  
2. 3.png — офіційний графічний логотип Науково-виробничого центру "Укрексперт" у форматі PNG.

## **🚀 Як запустити проект локально**

Якщо ви хочете переглянути тест на власному комп'ютері:

1. Завантажте файли index.html та 3.png.  
2. Покладіть їх **в одну папку**.  
3. Двічі клікніть на файл index.html — тест миттєво відкриється у вашому браузері з усіма зображеннями та стилями.

## **🌐 Швидкий безкоштовний деплой в інтернет (GitHub Pages)**

Цей проект повністю готовий до миттєвої публікації в мережі:

1. Створіть новий публічний (**Public**) репозиторій на GitHub.  
2. Завантажте туди файли index.html та 3.png.  
3. Перейдіть до **Settings** (Налаштування) \-\> **Pages** (Панель зліва).  
4. У розділі **Branch** виберіть гілку main (або master) та натисніть **Save**.  
5. Зачекайте 1 хвилину — у верхній частині екрана з'явиться активне посилання на ваш опублікований тест\!
